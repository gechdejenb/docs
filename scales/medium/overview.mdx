# Medium Scale Projects Overview

The **medium scale** is SkipSetup's comprehensive solution for building production-ready SaaS applications, startup platforms, and business tools. It includes everything you need to launch, scale, and manage a successful product from day one.

## What is Medium Scale?

Medium scale projects are **complete SaaS architectures** that include advanced features like admin dashboards, payment processing, monitoring systems, and enhanced security. Built for teams that need to ship professional products with enterprise-grade features.

## Quick Start Example

```bash
# Create your SaaS project
npx skipsetup-cli@latest create my-saas-platform --size medium

# Navigate to project
cd my-saas-platform

# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env
# Edit .env with your Stripe, database, and auth keys

# Set up database
pnpm db:push
pnpm db:seed

# Start development
pnpm dev

# Access admin dashboard at /admin
```

## When to Choose Medium Scale

<Columns cols={2}>
  <Card icon="check">
    <strong>Choose Medium Scale When:</strong>
    <ul>
      <li>You're building a revenue-generating product</li>
      <li>Payment processing is required from launch</li>
      <li>You need admin tools for customer support</li>
      <li>Analytics and monitoring are business-critical</li>
      <li>You have a team of developers collaborating</li>
      <li>Scalability and performance are priorities</li>
      <li>You need enterprise-grade security features</li>
    </ul>
  </Card>
  <Card icon="times">
    <strong>Consider Small Scale When:</strong>
    <ul>
      <li>You're validating a concept or idea</li>
      <li>Budget for infrastructure is very limited</li>
      <li>You're working solo on a side project</li>
      <li>You don't need payment processing yet</li>
      <li>Admin features can wait for later</li>
